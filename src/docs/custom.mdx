# Custom configuration

## Using a custom configuration

To use a custom config, import the **ThemeProvider** 
from Styled-components ```import { ThemeProvider } from 'styled-components'```, 
and pass a named object ```awesomegrid``` to the theme prop, as the example below.
The properties of your custom configuration will overwrite the default settings. Just set properties that are different from the default.

Try changing any property in customConf below and see what happens.

<Playground>
{() => {
  const customConf = {
    columns: 12,
    gutterWidth: 16, // px
    paddingWidth: 16, // px
    mediaQuery: 'only screen',
    container: {
      sm: 540, // px
      md: 720, // px
      lg: 960, // px
      xl: 1140 // px
    },
    breakpoints: {
      sm: 576, // px
      md: 768, // px
      lg: 992, // px
      xl: 1200 // px
    }
  }
  return (
    <ThemeProvider theme={{awesomegrid: customConf}}>
      <Container>
        <Row>
          <Col debug xs={4} sm={6} md={9}>
            <span role="img" aria-label="so cool">
              xs={1} sm={3} md={6} lg={6}
            </span>
          </Col>
          <Col debug noGutter>
            <Col debug>inside another Col</Col>
            <Col debug>inside another Col</Col>
          </Col>
        </Row>
        <Row>
          <Col debug xs={12} sm={6} md={10} lg={6}>
            xs={12} sm={6} md={10} lg={6}
          </Col>
          <Col debug xs={12} sm={6} md={2} lg={6}>
            xs={12} sm={6} md={2} lg={6}
          </Col>
        </Row>
      </Container>
    </ThemeProvider>
    )
  }}
</Playground>


## Default configuration

| prop         | type   | default value                           | description                                  |
|--------------|--------|-----------------------------------------|----------------------------------------------|
| columns      | number | 12                                      | number of columns on grid                    |
| gutterWidth  | number | 16                                      | gutter width in pixels                       |
| paddingWidth | number | 12                                      | size of outer padding on container in pixels |
| mediaQuery   | string | 'only screen'                           | media type of media query                    |
| container    | shape  | { sm: 540, md: 720, lg: 960, xl: 1140 } | size of containers in pixel                  |
| breakpoints  | shape  | { sm: 576, md: 768, lg: 992, xl: 1200 } | breakpoints in pixel                         |
